## 2. PERSONAS E JOBS-TO-BE-DONE

### Persona 1: Mãe/Pai (Primário)
- **Job:** "Quero acompanhar a rotina do meu bebê sem esquecer nada e sem esforço"
- **Dor principal:** Exaustão, insegurança, "será que meu bebê está mamando o suficiente?"
- **Trigger de download:** Nascimento (maternidade), indicação de pediatra, grupo de mães

### Persona 2: Pediatra (Canal + Usuário Indireto)
- **Job:** "Quero dados confiáveis sobre a rotina do bebê antes da consulta"
- **Dor principal:** Pais não trazem dados, consulta vira adivinhação
- **Motivação para indicar:** Consultas mais eficientes, imagem de modernidade, dados melhores

### Persona 3: Babá/Cuidador (Usuário Operacional)
- **Job:** "Quero registrar tudo que fiz para os pais verem e confiarem em mim"
- **Dor principal:** Relatório manual no WhatsApp, falta de confiança dos pais
- **Trigger:** Pais pedem para usar o app

### Persona 4: Creche/Berçário (B2B - futuro)
- **Job:** "Quero comunicar com os pais de forma profissional sobre o dia do bebê"
- **Dor principal:** Papel, WhatsApp caótico, falta de registro formal
- **Modelo:** Per-child fee ou licensing anual

## 4. CANAL ESTRATÉGICO: PEDIATRA

### Por Que o Pediatra é o Canal #1

1. **Confiança:** Mães confiam no pediatra mais do que em qualquer anúncio
2. **Timing perfeito:** Primeira consulta pós-alta = maior dor = maior conversão
3. **Recorrência:** Consultas mensais nos primeiros 12 meses = múltiplos touchpoints
4. **Custo zero de aquisição:** Pediatra indica por gerar valor na consulta

### Proposta de Valor para o Pediatra

#### Nível 1 — Indicação simples (MVP)
- Pediatra recomenda Yaya aos pais
- Pais exportam relatório PDF para levar na consulta
- **Entregável:** QR code para consultório + material explicativo A5
- **Incentivo:** Yaya+ gratuito para o pediatra (uso pessoal se tiver filhos) + selo "Recomendado por [Pediatra]"

#### Nível 2 — Relatório Pediatra (Fase 2)
- Relatório PDF profissional com dados de 30 dias:
  - Média diária: amamentações, volume mamadeira, fraldas, horas de sono
  - Gráficos de tendência semanal
  - Curva de peso/altura vs percentil WHO
  - Alertas automáticos (padrões fora do esperado)
- **Formato:** PDF A4 com branding Yaya + nome do pediatra
- **Entregável:** Botão "Preparar relatório para consulta" no app

#### Nível 3 — Dashboard Pediatra (Fase 3)
- Pediatra tem login próprio no Yaya
- Vê dados dos pacientes que autorizaram compartilhamento
- Recebe alertas de padrões anormais (queda de ingestão, sono irregular)
- **Modelo:** Gratuito para pediatra → pais precisam ser Yaya+ para compartilhar dados

### Roadmap do Canal Pediatra
| Milestone | Fase | Entregável |
|-----------|------|-----------|
| Material para consultório (QR code, folder A5) | Fase 1 | PDF imprimível + landing page |
| Relatório PDF para consulta | Fase 2 | Feature no app |
| Selo "Recomendado por Dr(a). X" | Fase 2 | Feature no perfil do pediatra |
| Dashboard pediatra web | Fase 3 | Portal web separado |
| API com prontuário eletrônico | Fase 3+ | Integração técnica |

## 5. CANAL ESTRATÉGICO: BABÁS E CUIDADORES

### Por Que Babás São Importantes

1. **Multiplier effect:** 1 babá pode trabalhar para 3-4 famílias ao longo do ano
2. **Advocacy natural:** "Eu uso o Yaya nas famílias que cuido" → recomendação forte
3. **Necessidade real:** Babás precisam dar satisfação aos pais — Yaya resolve isso
4. **Diferencial brasileiro:** No Brasil, babás são muito comuns vs EUA (onde daycare domina)

### Features para Babás/Cuidadores
| Feature | Descrição | Prioridade |
|---------|-----------|-----------|
| Perfil de cuidador | Tipo de acesso (babá, avó, creche), foto, horários | Alta |
| Relatório do turno | Resumo automático "Das 8h às 17h: 2 mamadas, 3 fraldas, 1h30 de sono" | Alta |
| Notificação para pais | Push quando cuidador registra atividade | Média |
| Permissões granulares | Pais controlam o que cuidador vê/edita | Média |
| Modo "handoff" | Ao trocar turno, transfere contexto (último registro, próxima previsão) | Baixa |
